## Como usar o projeto

- API
    - Crie seu cluster no mongoDB remoto
    - Adicione o .env de acordo com o .env.example
    - npm install
    - node .

- APP
    - Adicione 0 .env de acordo com o .env.example
    - npm install
    - npm run start
