Informações

Os campos dos formulários não são persistidos no banco de dados, são armazenados em localStorage e descartados ao recarregar a página.
Apenas os dados de preenchimento são armazenados no banco de dados.

O armazenamento se dá por combinação de chave-valor, a chave sendo o id do campo e o valor a resposta preenchida (ou escolhida, no caso de checkboxes).
